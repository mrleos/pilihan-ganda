## Cara Menggunakan
1. Clone repositori ini atau unduh file HTML
2. Buka file `index.html` dengan browser
3. Jawab semua pertanyaan
4. Lihat skor dan gambar pemenang saat selesai!

## Cara Menyesuaikan

    question: "Apa warna buah apel merah?",
